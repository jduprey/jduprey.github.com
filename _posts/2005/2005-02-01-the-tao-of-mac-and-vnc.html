The Tao of Mac is a pretty neat site for tech. stuff about the Mac.

I recently stumbled onto it, again, while looking for a better VNC client for my Mac. I didn't really find one. What I'd really like is remote desktop functionality like as it is provided in X windows or Windows remote Desktop. These technologies actually make remote access usable for long periods whereas VNC just never seems to be responsive enough.

Anywho.. here are some links I collected on VNC:
RealVNC - Official home of VNC
OSXvnc - VNC server for OS X
Chicken of the VNC - VNC client
VNC Client - another client that supports misc. compressions, and TightVNC server
MetaVNC - a window aware VNC
